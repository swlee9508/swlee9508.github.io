Caesar(시져) 방식의 암/복호화

시져 암/복호화 방식을 파이썬으로 구현 하기!!

Posted by SWL9508 on 11.20, 2019

Caesar(시져) 방식의 암/복호화 란??

간단한 대칭 암호 방식의 종류로 카이사르 암호라고 불리면서
각각의 알파벳을 정해진 숫자(key 값)만큼 이동시켜서 기존의 단어에서 치완된 단어를 반환 하는
암호화 방식을 이야기 합니다.

* 예를 들어 각각의 "Hello" 라는 알파벳 에서 3 만큼 이동하여 암호화를 진행하면 "Hello" -> "Jgoor" 로 변경 되어서 암호문이 생성 됩니다.*

Python Caesar_Cipher Test code 작성

아래의 화면 처럼 새로운 파이썬 파일을 생성 하여 진행 하도록 하겠습니다.
필자는 이름을 Caesar_Cipher 로 정하고 파일을 생성하겠습니다.

맨 처음 우리는 어떠한 평문 즉(메세지)를 암호화 시키기 위해 key를 먼저 정해 주어야 됩니다.
이번 실습에서는 영어의 대 소문자의 전체의 개수 값을 key 값으로 정의 하겠습니다.

그 개수를 알기 위해서 "string" 이라는 패키지 안에 있는 "ascii_letters" 라는
메서드를 이용하여 구할 수 있습니다.

첫번째 줄에 string 패키지를 불러오기 위해 import string 을 작성 하도록 하겠습니다.
그 다음 우리는 LETTERS 라는 변수 명을 가진 변수에 대소문자의 전체 길이를 저장 하겠습니다.
LETTERS = string.ascii_letters 을 통해 변수의 대소문자 전체 개수 값을 저장합니다.

아래의 코드를 작성 하시면 됩니다.

다음으로 암호화 할 메세지를 String 타입으로 작성 하여 줍니다.
이번 포스트에서는 "hello CaesarCipher" 라는 메세지를 암/복호화 하겠습니다.

메세지를 저장할 변수를 plainT 로 정의 하고 값을 저장 합니다.
plainT = "hello CaesarCipher"

평문 생성이 끝나면 우리는 암호화시 이용할 key 값을 생성해야 됩니다.
key 값을 생성 하기 위해 random 패키지를 import 시켜주겠습니다.

그 다음 key 라는 변수에 0~문자열 총 길이 사이에 랜덤인 숫자값을 하나 정하여서 저장 하겠습니다.

key = random.randrange(0,len(LETTERS))
randrange 는 랜덤을 돌릴 범위를 이야기 하고 len(LETTERS)는 LETTERS 의 전체 길이를 의미 합니다.
