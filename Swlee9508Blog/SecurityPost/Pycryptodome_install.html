PyCryptodome 설치 방법

파이썬 보안 관련 라이브러리 설치 하기!!

Posted by SWL9508 on 11.20, 2019

PyCryptodome 이란??

파이썬을 이용한 암/복호화 실습을 돕는 기능들이 구현 되어있는 보안관련 라이브러리 입니다.
이 라이브러리 에서는 다양한 실습 코드들과 RSA,AES 암호화 방식 등을 구현 가능하게 하는 다양한 기능들을 제공 합니다.

이번 포스팅에서는 이 프로그램을 설치되어 있는 파이썬에 추가하여 테스트 해보는 시간을 가져보도록 하겠습니다.

PyCryptodome 관련 API 페이지

1. 위 링크를 클릭하면 PyCryptodome 관련 Package 및 자료를 찾을 수 있으니 참고용으로 들어가서 보시면 도움이 될 것입니다.
(아래 사진은 링크 페이지 사진 입니다.)

2. Pychram 을 실행 시키시면 처음 화면에 저번 시간에 만든 파이썬 프로젝트 파일이 자동으로 열립니다.
(만약 저 화면이 안나오실 경우 open projcet 를 누르셔서 직접 프로젝트를 실행 시켜 주시면 됩니다.)

3. 위 화면이 정상적으로 나오신다면 다음으로 상단에
File -> Other Settings -> Preferences for New Project ->
Project Interpreter 를 클릭하시면
아래와 같은 화면이 나오는걸 확인 하실 수 있습니다.

4. 위 화면에서 필자는 이미 설치 되어 있는 많은 라이브러리들이 있지만.
처음 파이선을 설치 하시는 분들은 pip 와 pytz 두 가지만 설치가 되어 있을 것입니다.

pip 만 정상적으로 설치 되어 있으면 문제가 되지 않으니 오른쪽(맥 os는 아래)에 있는 + 버튼을 클릭하여서 라이브러리를 설치 해줍니다.

5. 다음으로 + 버튼을 선택 하시면 위 화면과 같이 검색해서 설치 할 수 있는 화면이 보이면 검색 창에 pycryptodome 을 입력 하시고 아래의 Install Package 버튼을 클릭하여 설치를
진행 하시면 됩니다.

Python IDE 설치

파이썬을 편하게 사용할 수 있는 여러가지 툴이 있지만 이번에는 JetBrains 사의 PyCharm tool 을 설치 방법을 설명 하겠습니다.

PyCharm 다운로드 홈페이지 링크

1. 위 링크를 클릭하여 PyCharm 다운로드 홈페이지로 접속 합니다.

2. 아래의 사진에 보이는 다운로드 버튼을 클릭해서 다운로드를 진행 합니다.
자신의 운영체제에 맞는 것을 선택합니다.(Community 버젼을 이용하면 무료로 사용 가능합니다.)

3. 아래의 화면이 보여지면서 다운로드가 시작 됩니다.

포스팅을 마무리 하면서.

파이썬 설치와 파이선을 쉽게 편집할 수 있는 JetBrains 사의 Pycharm 에 대한 설치 방법에 대해서 포스팅 해보았습니다. 이 글을 참고 하셔서
성공적인 파이썬 개발 환경 셋팅이 되길 바랍니다.
